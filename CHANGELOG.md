# Changelog

Все заметные изменения в этом проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-09

### Добавлено
- Полностью функциональный TUI интерфейс на Bubble Tea + Lipgloss
- Отображение статуса git репозитория с информацией о ветке и синхронизации
- Умное создание точек сохранения с предложениями описаний
- Просмотр истории точек сохранения с навигацией
- Откат к любой точке сохранения с hard reset
- Синхронизация с удалённым репозиторием (pull/push) с обработкой конфликтов
- Горячие клавиши для быстрых действий (C/H/R/S)
- Полная локализация интерфейса на русский язык
- Оптимизированное однобинарное распределение
- Автоматическая установка через install.sh
- Кросс-платформенная сборка для Linux, macOS, Windows

### Технические детали
- Go 1.21+
- Bubble Tea для TUI
- Lipgloss для стилизации
- go-git для git операций без системных зависимостей
